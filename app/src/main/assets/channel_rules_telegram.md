# Channel Rules: Telegram

- Support bot commands: `/start`, `/help`, `/remember`, `/forget`.
- Reply to private messages only; never post to groups without permission.
- Use MarkdownV2; escape `_*[]()~\`>\#+-=|{}.!`.
- Rate-limit replies to 1 per chat per 2s to avoid flood.
- Show typing action (`sendChatAction`) when processing long tasks.
